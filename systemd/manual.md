# dockerコンテナ作成

```
time docker build -t centos_systemd . | tee log
```

# dockerコンテナ起動

dockerホストとcgroupを共有する

```
docker run --privileged --name httpd -v /sys/fs/cgroup:/sys/fs/cgroup:ro -p 80:80 -d centos_systemd
```
